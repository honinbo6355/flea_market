### <사용한 기술>

- 프론트엔드 : html, css, javascript, jQuery, ajax, bootstrap, tiles
- 백엔드 : 전자정부 프레임워크, mysql, tomcat, MyBatis
- 외부 라이브러리 : Dropzone.js, Postcodify



<br />
